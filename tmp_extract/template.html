Report on the Event

Objectives:

1.To introduce the latest advancements in Capture The Flag (CTF) platforms and cybersecurity challenge design in the context of modern security practices.

2.To foster an understanding of the entrepreneurial strategies and innovations associated with building secure and scalable cybersecurity platforms.

3.To explore the practical applications and implications of CTF challenges such as OSINT, Web, and cloud security in various real-world scenarios.

4.To inspire participants to develop novel solutions and ideas in the field of cybersecurity and ethical hacking.

5.To encourage participants to engage in collaborative discussions and knowledge-sharing for the advancement of cybersecurity skills and practices.

Description of the event:

Round 1:

Session 1: Introduction to CTFQuest, overview of cybersecurity fundamentals, and briefing on platform architecture and competition guidelines.

Session 2: Hands-on setup of the CTF environment including deployment on local servers and cloud infrastructure.

Session 3: Training session on OSINT techniques, reconnaissance methods, and information gathering strategies.

Round2:

Session 1: Web security challenges covering common vulnerabilities such as injection flaws, authentication bypass, and misconfigurations.

Session 2: Cloud security and infrastructure session demonstrating deployment and management using Amazon Web Services EC2 instances and hybrid hosting.

Session 3: Live CTF competition with real-time challenge solving, monitoring, and scoring.

Round 3:

Session 1: Advanced challenges including privilege escalation, network analysis, and secure system design.

Session 2: Team collaboration, strategy discussions, and problem-solving workshops.

Session 3: Result announcement, feedback session, and knowledge-sharing with organizers and participants.
